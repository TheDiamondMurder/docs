---
title: "Terminate"
description: "The terminate feature"
---

The terminate feature terminates a user by reducing them to the lowest rank, you can do this immediately or after a given time period, and will automatically be put onto their record.

You can do this by using the command

```
/terminate [Roblox Username] (minutes to wait)
```

If you dont input the minutes til termination, it will be done instantly
