---
title: "Get Records"
description: "Fetches a users Records"
---

<Card title="GET">
  /v1/records/\{userID}
</Card>

## Inputs

<CodeGroup>
  ```json Headers
  token: "your-groups-token"
  guildid: "your-servers-id"
  ```

  ```json Params
  userID: "roblox-user-id"
  ```
</CodeGroup>

## Output

<CodeGroup>
  ```json 200
  {
    "records": [], // users records
    "username": "ExampleUser", // their username
  }
  ```
</CodeGroup>
