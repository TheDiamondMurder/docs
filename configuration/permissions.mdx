BloxPro makes configuring your server's permissions easy. You can add roles to 3 seperate categories: Owner, HICOM and Officer. You can also then configure what each category can do and what they have access to.

***

### Adding roles to permission[](#adding-roles-to-permission)

<Warning>
  Only the Discord Server owner can configure the servers permissions
</Warning>

It's pretty simple to add roles to a category

```
/perms addofficerrole [@role]
/perms addhicomrole [@role]
/perms addownerrole [@role]
```

Each of those commands will add the role to its respective category.

***

### Removing roles from a permission[](#removing-roles-from-a-permission)

If you want to remove a role from the permission category:

```
/perms removeofficerrole [@role]
/perms removehicomrole [@role]
/perms removeownerrole [@role]
```

***

### Configuring what each group can do[](#configuring-what-each-group-can-do)

<Warning>
  This is a BloxPro+ Premium Feature and requires a BloxPro+ Premium license
</Warning>

```
/perms changeperm [Permission Node] [Group]
```
